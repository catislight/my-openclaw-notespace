# Edge Case Catalog

## 目的

这个参考文件不是让你机械穷举，而是提醒你从不同维度补全“正常流程之外”的情况。

## 1. 输入维度

- 空值、缺字段、非法类型
- 超长输入、超短输入
- 特殊字符、emoji、多语言
- 重复提交、快速连续操作

## 2. 数据维度

- 空列表
- 单条数据
- 超大数据量
- 字段部分缺失
- 后端新增字段 / 删除字段 / 字段类型变化

## 3. 状态维度

- 初始态
- 加载态
- 成功态
- 失败态
- 重试态
- 中途中断态
- 部分成功态

## 4. 权限与角色维度

- 无权限
- 只读权限
- 权限动态变化
- 资源存在但当前用户不可操作

## 5. 时间与并发维度

- 连点
- 多请求竞争
- 先发后至
- 组件卸载后回调返回
- 时区差异
- 跨天/跨月/边界时间

## 6. 环境维度

- 开发 / 测试 / 生产配置差异
- 移动端 / 桌面端 / 小屏
- 浏览器差异
- 网络慢、断网、超时

## 7. 用户体验维度

- 空态是否可理解
- 错误提示是否可操作
- 失败后是否能恢复
- 加载反馈是否足够
- 禁用态是否明确
